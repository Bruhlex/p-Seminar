2. Firewall

In dieser vereinfachten Simulation wird der Verlauf bestimmter Signale angezeigt. Sie treffen mit unterschiedlichen Geschwindigkeiten auf die am Ende liegenden Sensoren. Ermittel wie hoch je die Wahrscheinlichkeit ist, das der Sensor getroffen wird. Mit Hilfe der beiliegenden Umrechnungstabelle kann das Passwort ermittelt werden. Die Wahrscheinlichkeiten sollen in Prozent umgewandelt werden.
• prozentuale Wahrscheinlichkeit = relative Häufigkeit in Prozent
• „Signal senden“ – Button -> um die Aufgabe zu starten
• „von vorne beginnen“ - Button -> um die Aufgabe neu zu starten
• erinnert euch an die absolute und relative Häufigkeit aus der siebten Klasse
• zur Umrechnungstabelle: z.B. 87% = 8 + 7 = KQ + CS = KQCS
